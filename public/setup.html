Why use OpenAI model names?

Cursor validates model names locally. By using standard OpenAI names (like gpt-4o), Cursor accepts the request. The proxy then automatically remaps:

gpt-4o → gemini-3-pro-high
gpt-4o-mini → gemini-3-flash
gpt-4 → claude-opus-4-5-thinking
gpt-3.5-turbo → gemini-3-flash
