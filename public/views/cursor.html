How Model Mapping Works

Cursor's free plan rejects unknown model names. We solve this by using OpenAI model names that Cursor accepts, then the proxy automatically remaps them:

gpt-4o → gemini-3-pro-high
gpt-4o-mini → gemini-3-flash
gpt-4.1 → claude-opus-4-5-thinking FREE
gpt-4 → claude-opus-4-5-thinking
gpt-3.5-turbo → gemini-3-flash

You can customize these mappings in Settings → Models tab. Yellow = works on Cursor free plan
